# Stack

The `Stack` component is intended for making vertical lists with evenly spaced elements.
It is functionally equivalent to using the `Flex` component with the `flexDirection` prop permanently set to `"column"`.
To control the spacing between childrem, use the `gap` prop which accepts any index valid in our theme's `spaces` scale.
