# Grid

The `Grid` component is an abstraction over CSS Grid Layout.
It accepts all of the CSS grid properties as props as well as any props that are accepted by the `Box` component.
I strongly recommend reading [the complete guide to CSS grid](https://css-tricks.com/snippets/css/complete-guide-grid/) to fully understand how to use this component.
