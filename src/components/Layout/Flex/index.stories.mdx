# Flex

The `Flex` component is just the `Box` component with the `display: flex;` property applied to it.
As a result, all the same rules of using `Box` apply to this component.
In addition, the `Flex` component accepts all the CSS flexbox properties as props.
I strongly recommend reading [the complete guide to CSS flexbox](https://css-tricks.com/snippets/css/a-guide-to-flexbox/) to fully understand how to use this component.
