# Inline

The `Inline` component is intended for making horizontal lists with evenly spaced elements.
It is mostly equivalent to using the `Flex` component with the `flexDirection` prop permanently set to `"row"`.
To control the spacing between childrem, use the `gap` prop which accepts any index valid in our theme's `spaces` scale.

What makes `Inline` unique is how it handles wrapping behavior.
If the component's children wrap, they will be spaced evenly both horizontally and vertically.
